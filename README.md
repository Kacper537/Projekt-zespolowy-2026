Aplikacja do planowania wydatków

Webowy system do kontrolowania finansów użytkownika.
Użytkownik wpisuje swoje wydatki i może zobaczyć:

* na co wydaje najwięcej,
* ile wydał w miesiącu,
* wykresy i statystyki,
* limity budżetu.


1. Rejestracja i logowanie

Użytkownik:

* zakłada konto,
* loguje się,
* ma własne dane i wydatki.


2. Dodawanie wydatków

Przykładowy formularz:

* kwota,
* kategoria,
* data,
* opis.

Kategorie:

* jedzenie,
* transport,
* rachunki,
* rozrywka,
* studia itd.


3. Lista wydatków

Tabela przykładowa:

| Data  | Kategoria | Kwota |
| ----- | --------- | ----- |
| 12.05 | Jedzenie  | 35 zł |
| 13.05 | Transport | 12 zł |

Możliwości:

* edycja,
* usuwanie,
* filtrowanie.


4. Budżety i limity

Np.

* limit na jedzenie = 800 zł,
* aplikacja pokazuje procent wykorzystania.


5. Dashboard

Strona główna:

* suma wydatków,
* ostatnie transakcje,
* wykres,
* aktualny budżet.

6. Technologie:
* PHP
* XAMPP
